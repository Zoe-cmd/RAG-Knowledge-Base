## 1. 概述

本文档为 AI 文档知识库（MVP）的高保真原型视觉说明。由于本项目由 AI Agent 设计，不产出 Figma 文件，因此以详细的文字描述、色值标注、组件规格和 Mermaid 图呈现高保真设计。前端工程师可基于本文档与 `docs/design-system.md`、`docs/wireframes.md` 直接实现。

**设计风格**：简洁现代、科技感、内容优先。参考 ChatGPT、Dify、MaxKB 等主流 AI 对话产品的视觉范式，保持 Element Plus 原生美感。

## 3. 流式回答高保真

### 3.1 流式效果描述

```
时间线：
t=0s     用户点击发送
         → 用户消息气泡出现（淡入动画 0.3s）
         → 输入框清空，发送按钮变灰
         → 对话区显示 "⟳ 检索中..." 旋转图标

t=0.5s   检索完成
         → 引用来源卡片出现（淡入 + 下滑 0.3s）
         → 卡片默认折叠，显示 "📄 引用来源 (3) ▼ 展开"

t=1s     首 token 到达
         → "检索中" 消失
         → AI 回答气泡出现（空内容）
         → 光标 "│" 开始闪烁（1s 循环）
         → 逐字追加 token

t=1~5s   流式生成中
         → 文字逐字显示
         → 光标在最新文字后闪烁
         → 发送按钮变为 "⏹ 停止生成"（红色）
         → 对话区自动滚动到底部

t=5s     回答完成
         → event: done
         → 光标消失
         → 回答底部显示 "⏱ 耗时 4.2s"（12px 灰色）
         → 停止按钮恢复为发送按钮
         → 输入框可用，聚焦
```

### 3.2 光标动画

```css
/* 流式光标 */
.streaming-cursor {
  display: inline-block;
  width: 2px;
  height: 1em;
  background-color: #409EFF;
  margin-left: 2px;
  vertical-align: text-bottom;
  animation: blink 1s infinite;
}

@keyframes blink {
  0%, 50% { opacity: 1; }
  51%, 100% { opacity: 0; }
}
```

### 3.3 Markdown 渲染

AI 回答支持 Markdown 渲染：
- **代码块**：深色背景 #1F2329，等宽字体，语法高亮
- **列表**：标准缩进，主色项目符号
- **表格**：边框 #E4E7ED，表头浅灰背景
- **链接**：主色 #409EFF，下划线
- **加粗**：字重 600
