## 关键用户场景

UI/UX 设计师需重点覆盖以下用户场景：

### 场景 1: 文档上传与知识库建立（核心闭环起点）

**用户旅程**：用户进入应用 → 看到空状态引导 → 拖拽/选择文档上传 → 看到上传状态流转（上传中→解析中→切分中→向量化中→完成）→ 文档出现在列表 → 知识库就绪可提问

**设计要点**：
- 空状态引导：无文档时提示"拖拽文档到这里开始建立知识库"
- 上传区域：支持点击选择与拖拽，拖拽时高亮
- 状态流转：每个文档显示当前处理状态，失败时显示原因与重试
- 文档列表：文档名、类型图标、大小、上传时间、切片数、状态

### 场景 2: RAG 问答（核心价值展示）

**用户旅程**：用户在对话区输入问题 → 看到加载状态 → 引用来源先显示 → 回答逐字流式显示 → 回答完成 → 可继续追问

**设计要点**：
- 对话区：左右分栏布局，右侧为对话区
- 引用来源：在回答上方或下方展示，含文档名、片段预览
- 流式渲染：逐字显示效果，首 token < 3s
- 停止按钮：流式过程中可点击停止
- 无相关内容：提示"未在文档库中找到相关内容"

### 场景 3: 多轮对话与历史管理

**用户旅程**：用户提问 → 获得回答 → 继续追问（上下文保留 4 轮）→ 切换到其他会话 → 查看历史 → 删除不需要的会话

**设计要点**：
- 会话列表：左侧显示，含标题（首条问题前 20 字）、消息数、最后活跃时间
- 会话切换：点击切换，当前会话高亮
- 多轮上下文：用户无感知（系统自动管理 4 轮截断）
- 历史回看：刷新后恢复当前会话历史
- 会话操作：新建、删除（二次确认）、清空（二次确认）

### 场景 4: 文档管理

**用户旅程**：用户查看文档列表 → 查看文档详情 → 删除过时文档（同步删除向量）

**设计要点**：
- 文档列表：分页展示，每页 20 条
- 删除确认：二次确认对话框
- 删除反馈：删除后列表即时更新

## 设计相关功能需求

### 布局需求

| 区域 | 内容 | 依据 |
|------|------|------|
| 左侧栏（上） | 会话列表 | US-017 |
| 左侧栏（下） | 文档列表/上传入口 | US-001/004 |
| 右侧主区 | 对话区（消息流 + 输入框） | US-010 |
| 顶部栏（可选） | 标题、知识库统计、Provider 显示 | US-009 |

### 交互状态设计

UI/UX 设计师需为以下状态设计视觉反馈：

| 状态 | 触发场景 | 设计要求 |
|------|----------|----------|
| 空状态-无文档 | 文档库为空 | 引导用户上传文档 |
| 空状态-无会话 | 无聊天会话 | 引导用户开始提问 |
| 上传中 | 文件上传 | 进度条/百分比 |
| 解析中 | 文档解析 | 加载动画 + "解析中"文字 |
| 切分中 | 文本切分 | 加载动画 + "切分中"文字 |
| 向量化中 | Embedding 生成 | 加载动画 + "向量化中"文字 |
| 检索中 | RAG 检索 | 加载动画 + "检索中"文字 |
| 流式回答中 | LLM 生成 | 逐字显示 + 光标闪烁 + 停止按钮 |
| 错误状态 | 任何失败 | 友好错误提示 + 重试按钮 |
| 无相关内容 | 检索结果为空 | 提示"未在文档库中找到相关内容" |

### 视觉要素

| 要素 | 要求 | 依据 |
|------|------|------|
| 界面语言 | 简体中文 | DEC-006 |
| 暗色模式 | V1 不做 | DEC-013 |
| 引用来源展示 | 文档名 + 片段预览，可点击定位 | US-012 |
| 流式渲染 | 逐字显示，首 token < 3s | US-011 |
| 文档类型图标 | PDF/Word/Markdown/TXT 各有图标 | US-004 |

### Element Plus 组件适配

PRD 建议使用以下 Element Plus 组件（技术栈 DEC-007）：

| 功能 | Element Plus 组件 | 依据 |
|------|-------------------|------|
| 文档上传 | el-upload（拖拽模式） | US-001/003 |
| 文档列表 | el-table + el-pagination | US-004 |
| 会话列表 | el-menu 或自定义列表 | US-017 |
| 对话区 | 自定义消息列表 | US-010 |
| 输入框 | el-input（textarea） + 发送按钮 | US-010 |
| 确认对话框 | el-message-box | US-005/018/019 |
| 状态标签 | el-tag | FR-DOC-007 |
| 加载状态 | el-skeleton / el-loading | 各加载状态 |
