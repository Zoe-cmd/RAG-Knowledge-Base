## 1. 设计原则

| 原则 | 说明 | 应用示例 |
|------|------|----------|
| 简洁优先 | 内容为王，去除冗余装饰 | 对话区无背景纹理，突出问答内容 |
| 一致性 | 视觉、交互、文案统一 | 所有主操作按钮使用主色；所有加载状态使用统一样式 |
| 清晰反馈 | 每个操作都有即时视觉反馈 | 上传时进度条；流式回答时光标闪烁 |
| 渐进呈现 | 先展示最重要的信息 | 引用来源默认折叠，点击展开 |
| 容错设计 | 允许错误并轻松恢复 | 删除前二次确认；错误可重试 |
| 性能感知 | 通过骨架屏优化等待体验 | 文档列表加载时显示骨架屏 |

## 3. 字体系统

### 3.1 字体族

```css
:root {
  --font-family-sans: 'PingFang SC', 'Microsoft YaHei', 'Helvetica Neue',
                      Helvetica, Arial, sans-serif;
  --font-family-mono: 'JetBrains Mono', 'Fira Code', 'Consolas',
                      'Courier New', monospace;
}
```

**说明**：优先使用系统中文字体（PingFang SC 用于 macOS，Microsoft YaHei 用于 Windows），避免额外加载 Web 字体影响性能。代码块使用等宽字体。

### 3.2 字号与行高

| 用途 | 字号 | 字重 | 行高 | 说明 | CSS 变量 |
|------|------|------|------|------|----------|
| 页面标题 | 20px | 600 | 28px | 顶部应用标题 | `--el-font-size-large` |
| 区块标题 | 16px | 600 | 24px | 卡片/面板标题 | `--el-font-size-medium` |
| 正文 | 14px | 400 | 22px | 默认正文、消息内容 | `--el-font-size-base` |
| 辅助文字 | 13px | 400 | 20px | 时间戳、切片数等 | 自定义 |
| 小字体 | 12px | 400 | 18px | 标签、元数据 | `--el-font-size-small` |
| 对话内容 | 15px | 400 | 24px | 聊天消息（略大便于阅读） | 自定义 |
| 代码块 | 13px | 400 | 20px | 代码、引用片段 | `--el-font-size-small` |

### 3.3 字重

## 7. 核心组件规范

### 7.1 按钮（Button）

基于 `el-button`，定义以下变体：

| 变体 | 样式 | 使用场景 | 示例 |
|------|------|----------|------|
| Primary | 主色背景 + 白字 | 主要操作 | 发送、上传 |
| Default | 白色背景 + 灰边框 | 次要操作 | 取消、关闭 |
| Text | 透明背景 + 主色字 | 文字操作 | 查看详情、展开 |
| Danger | 红色背景 + 白字 | 危险操作 | 删除、清空 |

**按钮状态**：

| 状态 | Primary | Default | 说明 |
|------|---------|---------|------|
| Default | #409EFF | #FFFFFF | 默认 |
| Hover | #337ECC | #F5F7FA | 鼠标悬停 |
| Active | #2664A6 | #E4E7ED | 按下 |
| Disabled | #A0CFFF | #F5F7FA | 禁用（opacity: 0.5） |
| Loading | 主色 + 旋转图标 | 灰色 + 旋转图标 | 加载中 |

**按钮尺寸**：

| 尺寸 | 高度 | 内边距 | 字号 | 用途 |
|------|------|--------|------|------|
| Small | 28px | 12px | 12px | 紧凑空间 |
| Default | 36px | 16px | 14px | 默认 |
| Large | 40px | 20px | 15px | 发送按钮、主 CTA |

### 7.2 输入框（Input）

基于 `el-input`：

| 类型 | 样式 | 用途 |
|------|------|------|
| 文本输入 | 高 36px，圆角 4px，灰边框 | 文档名搜索 |
| 文本域（textarea） | 最小高 80px，可自动扩展 | 聊天输入框 |
| 拖拽区域 | 虚线边框，高 200px，圆角 8px | 文档上传 |

**输入框状态**：

| 状态 | 边框色 | 说明 |
|------|--------|------|
| Default | #E4E7ED | 默认 |
| Hover | #C0C4CC | 悬停 |
| Focus | #409EFF + 阴影 | 聚焦（box-shadow: 0 0 0 2px rgba(64,158,255,0.2)） |
| Error | #F56C6C | 校验失败 |
| Disabled | #E4E7ED + 灰底 | 禁用 |

### 7.3 消息气泡（Chat Bubble）

自定义组件，非 Element Plus 原生：

**用户消息**：
- 对齐：右对齐
- 背景色：#ECF5FF（主色 50）
- 文字色：#303133
- 圆角：8px（右上角为 0，表示发送方）
- 内边距：12px 16px
- 最大宽度：70%
- 字号：15px

**AI 回答**：
- 对齐：左对齐
- 背景色：#FFFFFF
- 文字色：#303133
- 圆角：8px（左上角为 0）
- 内边距：12px 16px
- 最大宽度：80%
- 字号：15px
- 支持 Markdown 渲染（代码高亮、列表、表格）

**流式光标**：
- 样式：竖线 `│`
- 颜色：#409EFF
- 动画：闪烁（opacity 0 → 1 → 0，1s 循环）

### 7.4 引用来源卡片（Reference Card）

自定义组件：

- 背景：#F5F7FA（中性 100）
- 边框：1px solid #E4E7ED
- 圆角：6px
- 内边距：12px
- 位于 AI 回答上方或下方
- 默认折叠（显示前 2 条），点击"查看全部"展开
- 每条引用含：文档名（主色链接）、片段预览（2 行截断 + "..."）、来源路径

### 7.5 文档列表项（Document Item）

基于 `el-table` 行：

| 列 | 内容 | 宽度 | 说明 |
|----|------|------|------|
| 类型图标 | PDF/Word/MD/TXT 图标 | 40px | 不同颜色区分 |
| 文档名 | 文件名（超长省略） | 自适应 | 鼠标悬停显示完整名 |
| 大小 | 文件大小 | 80px | KB/MB 格式化 |
| 切片数 | chunk 数量 | 80px | 数字 |
| 状态 | 状态标签 | 100px | el-tag |
| 上传时间 | 日期时间 | 150px | YYYY-MM-DD HH:MM |
| 操作 | 删除按钮 | 80px | Danger Text 按钮 |

### 7.6 会话列表项（Session Item）

自定义列表项：

- 高度：60px
- 内边距：12px 16px
- 选中态：左侧 3px 主色边条 + 浅蓝背景
- 悬停态：浅灰背景
- 内容：标题（首条问题前 20 字，单行截断）、消息数 + 最后活跃时间（小字体灰色）

### 7.7 状态标签（Status Tag）

基于 `el-tag`：

| 状态 | 类型 | 文字 | 颜色 |
|------|------|------|------|
| 上传中 | warning | 上传中 | #E6A23C |
| 解析中 | warning | 解析中 | #E6A23C |
| 切分中 | warning | 切分中 | #E6A23C |
| 向量化中 | warning | 向量化中 | #E6A23C |
| 完成 | success | 完成 | #67C23A |
| 失败 | danger | 失败 | #F56C6C |

### 7.8 加载状态

| 场景 | 组件 | 样式 |
|------|------|------|
| 文档列表加载 | el-skeleton | 5 行骨架屏 |
| 会话列表加载 | el-skeleton | 3 行骨架屏 |
| 检索中 | el-loading | 对话区遮罩 + 旋转图标 + "检索中..." |
| 流式回答 | 光标闪烁 | 无遮罩，逐字显示 |
| 按钮加载 | el-button loading | 按钮内旋转图标 |

### 7.9 空状态

基于 `el-empty`：

| 场景 | 图标 | 文案 | 操作 |
|------|------|------|------|
| 无文档 | 文档图标 | "知识库还是空的，上传文档开始使用" | "上传文档"按钮 |
| 无会话 | 对话图标 | "开始你的第一个提问吧" | 无 |
| 无搜索结果 | 搜索图标 | "未找到匹配的文档" | 无 |
| 无相关回答 | 提示图标 | "未在文档库中找到相关内容" | "换个问题试试"提示 |

### 7.10 对话框（Dialog）

基于 `el-dialog`：

| 类型 | 标题 | 内容 | 按钮 |
|------|------|------|------|
| 删除确认 | "确认删除" | "确定要删除文档「{name}」吗？此操作将同步删除向量数据，不可恢复。" | 取消 + 确认删除（Danger） |

### 9.3 响应式设计

本项目以桌面端为主（本地 localhost 使用），定义 2 个断点：

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| Desktop | ≥ 1024px | 左右分栏完整布局（默认） |
| Tablet | < 1024px | 左侧栏可折叠为抽屉模式，通过按钮切换 |

**说明**：不做移动端适配（< 768px），因本地 localhost 场景以桌面端为主。最小推荐宽度 1024px。

## 10. 动效规范

### 10.1 过渡动画

| 场景 | 属性 | 时长 | 缓动函数 |
|------|------|------|----------|
| 按钮悬停 | background-color, color | 0.2s | ease |
| 卡片悬停 | box-shadow | 0.3s | ease |
| 对话框出现 | opacity, transform | 0.3s | ease-out |
| 列表项出现 | opacity, transform | 0.3s | ease-out |
| 抽屉展开 | transform | 0.3s | ease-in-out |
| 引用展开 | max-height | 0.3s | ease-in-out |

### 10.2 流式动效

| 场景 | 动效 | 说明 |
|------|------|------|
| 光标闪烁 | opacity 0 → 1 → 0 | 1s 循环 infinite |
| 逐字显示 | 无动画 | 直接渲染 token |
| 检索中 | el-loading 旋转 | 对话区遮罩 |

### 10.3 减少动画偏好

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

## 11. 无障碍设计

| 需求 | 实现方式 | 依据 |
|------|----------|------|
| 颜色对比度 | 文字与背景对比度 ≥ 4.5:1 | WCAG 2.1 AA |
| 键盘导航 | 所有交互元素可 Tab 聚焦 | WCAG 2.1 |
| 焦点可见 | 聚焦时显示 2px 主色 outline | WCAG 2.1 |
| 屏幕阅读器 | 图标加 aria-label，图片加 alt | WCAG 2.1 |
| 语义化 HTML | 使用语义标签（header, nav, main, aside） | 最佳实践 |

## 12. 文案规范

| 场景 | 文案 | 说明 |
|------|------|------|
| 上传成功 | "上传成功，已切分为 {n} 个片段" | 明确结果 |
| 上传失败 | "上传失败：{reason}" | 显示原因 |
| 解析失败 | "文档解析失败，请检查文件是否损坏" | 友好提示 |
| 扫描件提示 | "该 PDF 疑似扫描件，MVP 版本暂不支持 OCR，请上传文本型 PDF" | 明确限制 |
| 检索中 | "正在检索文档..." | 加载提示 |
| 流式中 | "正在生成回答..." | 加载提示（首 token 前） |
| 无相关内容 | "未在文档库中找到相关内容，请尝试换个问题或上传更多文档" | 引导操作 |
| 回答超时 | "回答生成超时，请重试" | 错误提示 |
| 删除确认 | "确定要删除「{name}」吗？此操作不可恢复。" | 二次确认 |
| 发送 | "发送" | 按钮 |
| 停止 | "停止生成" | 按钮 |
